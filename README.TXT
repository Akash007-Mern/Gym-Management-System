# 🏋️ Gym Management System

## 📌 Description

The **Gym Management System** is a Java-based application designed to manage gym members efficiently. It allows users to handle different types of memberships, maintain member records, and simulate a basic gym administration system.

---

## 🚀 Features

* Add and manage gym members
* Support for different membership types:

  * Regular Members
  * Premium Members
* Store and update member details
* Simple GUI interface (built using Java)
* Object-Oriented Design (OOP concepts used)

---

## 🛠️ Technologies Used

* Java
* BlueJ (for development and execution)

---

## 📂 Project Structure

* `GymGUI.java` → Main GUI of the application
* `GymMember.java` → Base class for members
* `RegularMember.java` → Regular membership class
* `PremiumMember.java` → Premium membership class

---

## ▶️ How to Run

1. Open the project in **BlueJ**
2. Compile all classes
3. Run the `GymGUI` class
4. Use the interface to manage members
